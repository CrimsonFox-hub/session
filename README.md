======================================================================

                         ТЕЛЕГРАМ-БОТ ДЛЯ АНАЛИЗА АКЦИЙ
                         
======================================================================

Бот для прогнозирования цен акций и генерации торговых рекомендаций на основе 
анализа временных рядов.

--------------------------------------------------------------------------------
                             БЫСТРЫЙ СТАРТ
--------------------------------------------------------------------------------

Windows:
1. Запустите install.bat
2. Настройте токен в файле .env
3. Запустите run.bat

Linux/Mac:
1. Дайте права на выполнение: chmod +x install.sh run.sh
2. Запустите: ./install.sh
3. Настройте токен в файле .env
4. Запустите: ./run.sh

--------------------------------------------------------------------------------
                               НАСТРОЙКА
--------------------------------------------------------------------------------

1. Получите токен бота у @BotFather в Telegram
2. Создайте бота командой /newbot
3. Скопируйте токен
4. В файле .env замените your_actual_bot_token_here на ваш токен

--------------------------------------------------------------------------------
                              ИСПОЛЬЗОВАНИЕ
--------------------------------------------------------------------------------

Основные команды:\
• /start    - начать работу с ботом\
• /tickers  - показать доступные тикеры для анализа\
• /help     - помощь по использованию бота

Анализ акций:\
Отправьте боту сообщение в формате:\
ТИКЕР СУММА

Примеры:\
AAPL 1000\
TSLA 50000\
SIBN.ME 100000

--------------------------------------------------------------------------------
                            ФУНКЦИОНАЛЬНОСТЬ
--------------------------------------------------------------------------------

• Прогноз цен акций на 30 дней вперед\
• Использование 4 моделей машинного обучения (Ridge, Random Forest, ARIMA, LSTM)\
• Автоматический выбор лучшей модели по метрикам качества\
• Генерация торговых рекомендаций (точки покупки и продажи)\
• Расчет потенциальной прибыли в рублях с учетом курса USD/RUB\
• Визуализация прогноза с графиком и торговыми точками\
• Подробное логирование всех запросов и результатов

--------------------------------------------------------------------------------
                               ТЕХНОЛОГИИ
--------------------------------------------------------------------------------

• Python 3.7+\
• TensorFlow для LSTM моделей\
• scikit-learn для классических ML моделей\
• statsmodels для статистических моделей (ARIMA)\
• yfinance для загрузки данных акций\
• python-telegram-bot для телеграм-бота\
• matplotlib для визуализации

--------------------------------------------------------------------------------
                           АРХИТЕКТУРА ПРОЕКТА
--------------------------------------------------------------------------------

Проект использует модульную архитектуру:

bot/
├── core.py          - основной класс бота\
├── models.py        - управление ML моделями\
├── trading.py       - логика торговых стратегий\
└── visualization.py - построение графиков

utils/
├── data_loader.py   - загрузка данных\
└── helpers.py       - вспомогательные функции

config.py           - конфигурация приложения\
main.py             - точка входа\
run.py              - скрипт запуска
