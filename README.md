Телеграм-бот для анализа акций
Бот для прогнозирования цен акций и генерации торговых рекомендаций на основе анализа временных рядов.

Быстрый старт
Windows:
Запустите install.bat

Настройте токен в файле .env

Запустите run.bat

Linux/Mac:
Дайте права на выполнение: chmod +x install.sh run.sh

Запустите: ./install.sh

Настройте токен в файле .env

Запустите: ./run.sh

Настройка
Получите токен бота у @BotFather в Telegram

Создайте бота командой /newbot

Скопируйте токен

В файле .env замените your_actual_bot_token_here на ваш токен

Использование
Основные команды:
/start - начать работу с ботом

/tickers - показать доступные тикеры для анализа

/help - помощь по использованию бота

Анализ акций:
Отправьте боту сообщение в формате:

text
ТИКЕР СУММА
Например:

text
AAPL 1000
TSLA 50000
SIBN.ME 100000
Функциональность
Прогноз цен акций на 30 дней вперед

Использование 4 моделей машинного обучения (Ridge, Random Forest, ARIMA, LSTM)

Автоматический выбор лучшей модели по метрикам качества

Генерация торговых рекомендаций (точки покупки и продажи)

Расчет потенциальной прибыли в рублях с учетом курса USD/RUB

Визуализация прогноза с графиком и торговыми точками

Подробное логирование всех запросов и результатов

Поддерживаемые тикеры
Американские акции:
AAPL (Apple), MSFT (Microsoft), GOOGL (Google), AMZN (Amazon)

TSLA (Tesla), META (Meta), NVDA (NVIDIA), JPM (JPMorgan Chase)

И многие другие

Российские акции:
SIBN.ME (Газпром), MTSS.ME (МТС), YNDX.ME (Yandex)

ETF и индексы:
SPY (S&P 500 ETF), QQQ (Nasdaq 100 ETF), VOO (Vanguard S&P 500)

Технологии
Python 3.7+

TensorFlow для LSTM моделей

scikit-learn для классических ML моделей

statsmodels для статистических моделей (ARIMA)

yfinance для загрузки данных акций

python-telegram-bot для телеграм-бота

matplotlib для визуализации

Архитектура проекта
Проект использует модульную архитектуру:

bot/core.py - основной класс бота

bot/models.py - управление ML моделями

bot/trading.py - логика торговых стратегий

bot/visualization.py - построение графиков

utils/data_loader.py - загрузка данных

utils/helpers.py - вспомогательные функции

Важно
Это учебный проект, созданный для демонстрации возможностей анализа временных рядов. Не используйте его для принятия реальных инвестиционных решений. Рынки акций волатильны, и прошлые результаты не гарантируют будущую доходность.
